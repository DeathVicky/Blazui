# blazui

#### 介绍
Element的blazor版本

Element 地址：https://element.eleme.cn/#/zh-CN/component/layout

Blazui 演示地址：http://47.103.163.62:9000/button

#### 使用前提
参考Blazor使用的前提条件

1. 安装 .Net Core 3.0
2. 安装 VS2019
3. 安装所有 VS2019 Blazor Extension

#### 使用说明

拉取代码，用 VS2019 打开，直接启动 Blazui.ServerRender 项目，组件残缺，暂时没有 nuget 包
